# Reproduction d'un Site Vitrine

Ce référentiel GitHub contient une reproduction d'un site vitrine, réalisée avec des connaissances de base en HTML/CSS. L'objectif de ce projet est de créer une version simplifiée d'un site web pour mettre en pratique les compétences en développement web.

![Aperçu du site](https://cdn.discordapp.com/attachments/1163380311887990784/1164563938910933033/Molengeek_1.png)

## Caractéristiques du Projet

- **Langages Utilisés**: HTML, CSS

## Présentation du Site

Le site reproduit ici est une version simplifiée d'un site vitrine. Il s'agit d'un exemple d'utilisation de HTML pour structurer le contenu de la page et de CSS pour la mise en forme. Le site peut servir de point de départ pour ceux qui souhaitent apprendre les bases du développement web.
